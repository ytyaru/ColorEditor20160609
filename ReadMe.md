# このソフトウェアについて #

ColorEditor20160609は私個人が学習目的で作成したソフトウェアです。

色を編集するアプリケーション案のモックアップ集です。

# 開発環境 #

* Windows XP SP3
* .NET Framework 4.0
* Windows Form Application
* C#
* Visual C# 2010 Express
